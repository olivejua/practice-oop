# 객체지향 설계 연습

## 객체지향 설계 방법
1. 책임-주도 
2. 디자인패턴
3. TDD

### 객체지향 세계에서는
- 역할과 책임, 협력 안에서 객체를 설계한다.
- 협력 안에서는 메시지를 전달할 뿐 어떻게 할지는 메시지 수신자가 자율적으로 판단한다.
- 행동이 먼저고 행동의 필요에 의해서 상태를 결정한다.
- 비슷한 행동이 보일 경우 그룹으로 단순화한다.

## TDD 리듬
- 테스트 목록 작성한다.
- 작은 단위부터 시작한다.
- 리팩토링하면서 필요한 테스트 목록을 하나씩 추가하면서 완성한다.
